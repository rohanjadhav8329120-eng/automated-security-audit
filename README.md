# 🔒 Automated Security Audit & Intrusion Detection System

## 📄 Overview
An automated security tool designed to analyze network logs, detect unauthorized access attempts, verify authorized connections, and generate professional compliance reports. This tool helps security teams identify threats in real-time and maintain audit trails.

Built as a demonstration of efficient data handling using Python set operations for O(1) lookup complexity, ensuring rapid analysis even with large datasets.

## 🚀 Features
- **Intrusion Detection**: Identifies and flags unauthorized IP addresses attempting to access the system.
- **Access Verification**: Validates active connections against an authorized whitelist.
- **Inactive Account Auditing**: Flags authorized accounts with no activity for security review.
- **Professional Reporting**: Generates clean, formatted reports with actionable intelligence and dynamic security status alerts.
- **Compliance Ready**: Output format suitable for security audits and regulatory compliance.

## 🛠️ Technologies Used
- **Python 3.x**: Core programming language.
- **Data Structures**: Advanced use of Sets for efficient O(1) lookup operations.
- **Security Protocols**: Implementation of basic security auditing logic.
- **System Tools**: Time-based logging and report generation.
